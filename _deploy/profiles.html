Las Mentes Detrás de la Tecnología

Conoce a los visionarios que crearon las tecnologías que usas todos los días

LT

Linus Torvalds

Creador de Linux y Git

Finlandia 🇫🇮

Ingeniero de software finlandés-estadounidense que creó el kernel de Linux y el sistema de control de versiones Git. Su trabajo alimenta la mayoría de servidores, smartphones y supercomputadoras del mundo.

Logros Clave

Creó el kernel de Linux (1991)Creó Git (2005)Premio Pionero IEEEPremio Tecnología del Milenio

Dato curioso: Linux ejecuta el 96.3% de los 1 millón de servidores web más importantes del mundo

Open Source
BG

Bill Gates

Cofundador de Microsoft

Estados Unidos 🇺🇸

Empresario, desarrollador de software y filántropo estadounidense. Cofundó Microsoft, que se convirtió en la empresa de software para computadoras personales más grande del mundo.

Logros Clave

Cofundó MicrosoftCreó MS-DOS y WindowsPersona más rica (varios años)Fundación Bill y Melinda Gates

Dato curioso: Microsoft Windows se ejecuta en más de 1,400 millones de dispositivos

GV

Guido van Rossum

Creador de Python

Países Bajos 🇳🇱

Programador neerlandés que creó el lenguaje de programación Python en 1991. Python es ahora uno de los lenguajes más populares y versátiles del mundo.

Logros Clave

Creó Python (1991)BDFL de Python por 30 añosIngeniero Senior de GoogleIngeniero Técnico de Microsoft

Dato curioso: Python es #1 en el índice TIOBE y potencia la IA/ML en todo el mundo

BE

Brendan Eich

Creador de JavaScript

Estados Unidos 🇺🇸

Programador estadounidense que creó JavaScript en 1995 mientras trabajaba en Netscape. También cofundó el proyecto Mozilla y creó el navegador Brave.

Logros Clave

Creó JavaScript (1995)Cofundó MozillaCreó Brave BrowserPremio Brendan Eich

Dato curioso: JavaScript se ejecuta en cada navegador web del mundo — el 97% de todos los sitios web

DR

Dennis Ritchie

Creador de C y Unix

Estados Unidos 🇺🇸

Científico informático estadounidense que creó el lenguaje de programación C y codesarrolló el sistema operativo Unix en Bell Labs. Su trabajo sentó las bases de la computación moderna.

Logros Clave

Creó el lenguaje CCodesarrolló UnixPremio Turing (1983)Medalla Nacional de Tecnología

Dato curioso: El lenguaje C es la base de la mayoría de sistemas operativos, bases de datos y compiladores

JG

James Gosling

Creador de Java

Canadá 🇨🇦

Científico informático canadiense que creó el lenguaje de programación Java en 1995 en Sun Microsystems. Java se ejecuta en miles de millones de dispositivos en todo el mundo.

Logros Clave

Creó Java (1995)Padre de JavaMedalla de Honor IEEEPremio ACM a Sistema de Software

Dato curioso: Java se ejecuta en 3,000 millones de dispositivos — desde teléfonos Android hasta servidores empresariales

AL

Ada Lovelace

Primera Programadora

Reino Unido 🇬🇧

Matemática británica que escribió el primer algoritmo destinado a ser procesado por una máquina. Es ampliamente considerada la primera programadora de la historia.

Logros Clave

Primer algoritmo de computaciónVisionaria de la computaciónLenguaje Ada nombrado en su honorPionera de la informática

Dato curioso: Sus notas sobre el motor de Babbage contenían el primer algoritmo publicado — en 1843

Empresario de medios y emprendedor de internet estadounidense que cofundó Facebook (ahora Meta). Transformó las redes sociales y el desarrollo web.

Creó Facebook/MetaMeta Quest VRReact de código abiertoLa plataforma social más grande

Desarrollador de software estadounidense que creó Node.js en 2009, revolucionando el JavaScript del lado del servidor. Después creó Deno como un runtime mejorado.
